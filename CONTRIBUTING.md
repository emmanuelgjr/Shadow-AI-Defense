# Contributing

Especially valued: detection engineers with field experience submitting
battle-tested rules, runbook improvements rooted in real incidents, and
honest tool / service annotations.

See [DETECTION_QUALITY.md](DETECTION_QUALITY.md) for the rule-acceptance
standard.

## Local checks

```bash
npm ci
npm run validate
npm run test
npm run build
```

All four must pass.
