# Templates and comms

This directory hosts downloadable Word templates referenced by `/comms` and
`/policy` pages.

In v0.1, files are stubs; the full template content lands progressively in
v0.2 as each passes review. Filename convention matches the comms IDs in
`/src/pages/comms.astro`.

The full Shadow AI policy text lives in `/src/pages/policy.astro` and is
displayed inline (it does not require download for v0.1).
